Ok what really is this?
A Minecraft server with grief mitigation.

Note that I italicize mitigation. It is hard to have the freedom of playing multiplayer vanilla Minecraft without a tradeoff with "grief security".

The core idea is that players should be able to build and have that build be off limits to other players.

One obvious problem is that you could build around a build and that sucks.
Teleportation will be added to mitigate trapping, but the actual result of this core idea is more so that players can adventure and build sealed off bases to store treasure.
